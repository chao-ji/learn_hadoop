Performs matrix addtion/substraction

Input:
  -Matrix A
    A1  a b c
    A2  d e f
  -Matrix B
    B1  g h i
    B2  j k l
    
Output:
  -Matrix C
    C1  a+/-g   b+/-h   c+/-i
    C2  d+/-j   e+/-k   f+/-l
    
hadoop jar MatrixAddition.jar MatrixAddition <newMatrixID> <sign> <input> <output>
